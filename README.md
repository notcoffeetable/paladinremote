paladin-remote
=============

Summary
-------
Remote control for Telestream's Wirecast through a node.js served web interface. OLE manipulation through Perl. Documentation in `/docs` by docco (currently incomplete).

Dependencies
------------
- Express
- mdns

How to Use/Install
----------
- Clone repo
- `cd paladinremote && npm install`
- `node server.js [-v]`)
